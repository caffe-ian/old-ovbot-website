Privacy Policy

For OV Bot Users

This policy states what data do we store when you use OV Bot. It also states the reason we store the data and how long do we store it.
For the privacy policy below, "them" refers to the data stored.

1  What data do we store?

We do not store any messages or files. The only data we store are
Users
    Username and tag
    User ID
Servers
    Server name
    Server ID
All data stored are automatically encrypted and secured.

2 Who can access them?

Only the owner of the bot can have access to the database and see the stored data.

3 Why do we store them?

Your Discord user ID will be used to load the rest of your data from the economy bot.
Your Discord username and tag is used for displaying in your profile.
Server ID is used for storing server settings such as disabled commands while the server name is used for displaying purposes only.

4 How long do we store them?

Users data will be stored forever in our database, even if you are banned from the bot. For servers, data will be automatically deleted when the bot leaves the server. You can request for user data deletion, read below for more information.

5 How can I request a deletion of my user data?

You can send an email to us here requesting for user data deletion. Another option is to join our official server here and seek for support in the server.
